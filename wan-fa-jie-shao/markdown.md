# 二.河马NFT

进入【NFT商城】 或底部【NFT】进入河马购买页面。

同一高级河马 NFT 每人最多可购买 2 只。

购买高级 NFT 可获得 宠物养成加速卡（首次购买额外赠送，加速 20%宠物养成进度）。
